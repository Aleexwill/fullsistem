# API Reference

## Autenticación

### POST /api/v1/auth/login
Login de usuario

### GET /api/v1/auth/me
Obtener usuario actual

## Tickets

### GET /api/v1/tickets
Listar tickets

### POST /api/v1/tickets
Crear ticket

### GET /api/v1/tickets/{id}
Obtener ticket

### PATCH /api/v1/tickets/{id}
Actualizar ticket

### PATCH /api/v1/tickets/{id}/status
Cambiar estado

## Clientes

### GET /api/v1/clients
Listar clientes

### POST /api/v1/clients
Crear cliente
